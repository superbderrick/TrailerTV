# SummerPlayer
- SummerPlayer is simple videoPlayer app(Support Android and ios Both).
- It will be used Android(Exoplayer) and iOS(AVPlayer API)with swift 3.0.
- iOS (ios9) Android (A kitkat OS version or higher is recommended.)


# Using open sources.

	iOS 
		UI :XLPagerTabStrip.

	Android
		Player :ExoPlayer.
